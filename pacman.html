It's Rainin' PacMen

Create as many PacMen as your heart desires with the PacMan Factory!

The PacMan Factory project pulls from an array of images to produce a single, random PacMan every time the user clicks Add PacMan. When the user clicks the Start Game button, the PacMen will begin moving at random velocities around the screen. Edge detection ensures that they never fall off!
